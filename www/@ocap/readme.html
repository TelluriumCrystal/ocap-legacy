Operation Capture And Playback
Addon v0.6.0

Installation

  1. Copy '@ocap' and 'userconfig' to your Arma 3 root directory.
  2. Navigate to A3 root > userconfig > ocap. Open config.hpp in a text editor (e.g. notepad++). Configure settings as desired.
Note: This mod is intended to be run server-side only. Clients do not need it installed.


Running OCAP

To run OCAP on your server, add @ocap to your server's '-mod' startup parameter.
e.g. -mod=@ocap;

Capture automatically begins when server becomes populated (see userconfig for settings).
To end and export capture data, call the following (server-side):
[] spawn ocap_fnc_exportData;

Tip: You can use the above function in a trigger.
e.g. Create a trigger that activates once all objectives complete. Then on activiation:
if (isServer) then { _export = [] spawn ocap_fnc_exportData; }; "end1" call BIS_fnc_endMission; // Ends mission for everyone


Note: For every 10 'frames' captured, OCAP will print a line in the Arma RPT log file. When exporting, the OCAP .dll extension also creates its own log file in the Arma server root folder called ocap_log.txt - use this when debugging.
